Incidence rate

RES_IR_benefit_fup_BC

Stratified table by sex, birth cohort, dose, type vax and week of followup. Contains the count and incidence rate with upper and lower bound for covid severity





RES_IR_benefit_week_BC

Stratified table by sex, birth cohort, dose, type vax and calendar week. Contains the count and incidence rate with upper and lower bound for covid severity





RES_IR_risk_fup_BC

Stratified table by sex, birth cohort, dose, type vax and week of followup. Contains the count and incidence rate with upper and lower bound for AESI, narrow and broad





RES_IR_risk_week_BC

Stratified table by sex, birth cohort, dose, type vax and calendar week. Contains the count and incidence rate with upper and lower bound for AESI, narrow and broad





RES_IR_benefit_fup_RF

Stratified table by sex, risk factors, dose, type vax and week of followup. Contains the count and incidence rate with upper and lower bound for covid severity





RES_IR_benefit_week_RF

Stratified table by sex, risk factors, dose, type vax and calendar week. Contains the count and incidence rate with upper and lower bound for covid severity





RES_IR_risk_fup_RF

Stratified table by sex, risk factors, dose, type vax and week of followup. Contains the count and incidence rate with upper and lower bound for AESI, narrow and broad





RES_IR_risk_week_RF

Stratified table by sex, risk factors, dose, type vax and calendar week. Contains the count and incidence rate with upper and lower bound for AESI, narrow and broad





RES_IR_week

Stratified table by sex, ageband, month, year and at_risk_at_study_entry. Contains the persontime, counts and incidence rate with upper and lower bound for each AESI
